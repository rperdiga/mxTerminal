General

Tab persistence and the buffer sizes that govern how much terminal output is kept in memory and on screen.

Shell

The shell each new tab spawns. Existing tabs aren't affected; close and reopen a tab to apply changes.

Studio Pro MCP

Studio Pro 11.10+ exposes its model-tier API as an MCP server. Enable to wire it into the CLIs you use inside this terminal — the URL we write into .mcp.json always tracks Studio Pro's own port (no manual port to keep in sync).

CLIs to wire

Action bridge

Exposes Studio Pro UI actions (run, stop, refresh from disk, save all, get app status) to the CLIs above as MCP tools — so Claude Code can drive Studio Pro the way you do.

Skills

Install prescriptive skill packs that teach Studio Pro patterns it doesn't ship with — files written into your Mendix project so Studio Pro can use them immediately.
